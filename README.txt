Jackcess Encrypt

Jackcess Encrypt is an extension library for the Jackcess project which
implements support for some forms of Microsoft Access and Microsoft Money
encryption. Jackcess Encrypt is licensed under the Apache License (as of
version 2.1.0). 


Please note that the GitHub repository is a mirror of the main project
repository which is hosted on SourceForge:

Homepage: https://jackcessencrypt.sourceforge.io/

Project: https://sourceforge.net/projects/jackcessencrypt/
